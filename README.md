# Dev Container

## Features

- Non-root user
- Zsh
- Oh my zsh
- Git
- Tmux
- Chezmoi
- Starship Prompt
- Neovim
- Rust
- Python
- Node
- Yarn
- Dotnet

## Usage

The first time, run `./setup.sh` to build the image and create the container.\
The subsequent times, run `./run.sh` to start the container and attach to it.
